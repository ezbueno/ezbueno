## 🚀 Featured Projects

### 🏆 [ChurnInsight Platform - ML Prediction API](https://github.com/ezbueno/churninsight-platform)
**Tech Stack:** Java, Spring Boot, ONNX Runtime, Machine Learning, Logistic Regression, SMOTE, Docker, Flyway, Prometheus  
📋 **Description:** Machine Learning API for predicting user churn in music streaming platforms. Features real-time inference using ONNX Runtime, batch processing (up to 1M records), AI-powered diagnostics, and advanced filtering with 15+ parameters. Built with Hexagonal Architecture for complete decoupling.  
✨ **Highlights:**  
- 🤖 Real-time ML inference with ONNX Runtime  
- 📊 Batch processing (CSV/XLSX) with live status tracking  
- 🧠 AI diagnostics analyzing risk and retention factors  
- 🏗️ Hexagonal Architecture (Ports & Adapters)  
- ⚡ High-performance: 10K+ req/s with parallel processing  
- 🔒 Rate limiting with bucket4j (IP/user-based)  
- 📈 Custom metrics via Actuator/Prometheus  
- 🐳 Full Docker containerization with Docker Compose  

🔗 **[View Repository](https://github.com/ezbueno/churninsight-platform)** | 🏆 **Hackathon ONE (Oracle Next Education) Project - Team DataBeats**
